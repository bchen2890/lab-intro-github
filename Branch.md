# lab-intro-github
### Create a new branch
```
$ git branch test
```
### List existing branches
```
$ git branch
```

### Switch to a branch
```
$ git checkout test
```

### Create and switch to a new branch
```
$ git checkout -b test
```

### Merge a branch into main
```
$ git checkout main
$ git merge test
```

### Delete a branch 
```
$ git branch -d test
```
